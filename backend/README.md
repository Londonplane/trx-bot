# 后台管理系统

## 概述
待开发的TRON能量供应商后台管理系统，提供API服务支撑Telegram Bot的业务功能。

## 技术栈
- FastAPI (Python异步Web框架)
- PostgreSQL (主数据库)
- Redis (缓存和任务队列)  
- Celery (异步任务处理)

## 核心功能
- 订单管理: 处理能量租赁订单的完整生命周期
- 钱包池管理: 管理供应商钱包和Energy库存
- 用户账户: 真实的余额管理和充值处理
- 财务系统: 收入成本核算和定价策略
- 监控报警: 系统状态和业务指标监控
